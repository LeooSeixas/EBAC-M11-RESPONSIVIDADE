# SOBRE O PROJETO

Este é um projeto desenvolvido para praticar conceitos avançados de Design Responsivo, CSS Grid, Flexbox e Unidades Relativas. O desafio consiste em criar a página de uma agência de design digital chamada "Agência Criativa Web", garantindo que a interface seja moderna, atrativa e totalmente adaptável a qualquer tamanho de tela (Desktop, Tablet e Mobile).

## OBJETIVO:
- Desenvolver uma Landing Page completa que utilize as melhores práticas de layout moderno. O foco principal é a estruturação de elementos complexos usando Grid e Flexbox

## Arquivos obrigatórios:
- INDEX.HTML: Estrutura semântica da aplicação;
- ESTILOS.CSS: Estilização completa, uso de Media Queries e variáveis de layout.

## Funcionalidades exigidas:
- Estrutura HTML:
    - Utilizar a estrutura básica de um documento HTML5 (!DOCTYPE html, html, head, body).
    - Criar um cabeçalho (header) contendo o logo da agência (fictício) e um menu de navegação (nav).
    - Criar as seguintes seções (section):
        - Home: Um banner de boas-vindas destacando a agência e seu diferencial;
        - Sobre Nós: Breve descrição da agência e seus valores;
        - Serviços: Lista de serviços oferecidos (ex: design gráfico, desenvolvimento web, branding, etc.);
        - Depoimentos: Comentários de clientes fictícios para reforçar a credibilidade da agência;
        - Contato: Informações para contato (endereço, telefone, e-mail e um formulário de contato).
    - Estilos CSS:
        - Criar um arquivo CSS externo chamado "estilos.css";
        - Aplicar cores, fontes e espaçamentos coerentes com uma identidade visual moderna e profissional;
        - Utilizar Flexbox para alinhar itens do menu de navegação e outras seções;
        - Utilizar CSS Grid para estruturar a seção "Serviços" em um layout responsivo;
        - Usar unidades relativas (%, vh, vw, rem, em, fr) para garantir flexibilidade no design;
        - Criar um menu responsivo que se ajuste para telas menores (hambúrguer menu opcional);
        - Garantir que imagens e textos sejam responsivos.
    -  Funcionalidade e Responsividade:
        - O site deve ser totalmente responsivo e adaptável para diferentes tamanhos de tela;
        - As imagens devem ser otimizadas para responsividade usando srcset;
        - O layout deve ser adaptado com media queries (@media) para diferentes resoluções;
        - O formulário de contato deve conter campos básicos (input para nome, e-mail e mensagem).

### Adicionais:
- Integração com Swiper JS: Carrossel interativo na seção de depoimentos para melhor experiência do usuário;
- Imagens Otimizadas: Uso de srcset na imagem principal para carregar diferentes resoluções baseadas na tela do dispositivo;
- Botão Scroll-to-Top: Funcionalidade em JavaScript para retornar ao topo da página suavemente;
- Header Sticky/Glassmorphism: O menu muda de estilo e fixa-se no topo ao realizar o scroll;
- Animações de Interface: Efeitos de hover em botões e cards, além de transições suaves em todos os elementos interativos;
- Design Mobile-First: Layout pensado primeiro para dispositivos menores, garantindo usabilidade total em smartphones.

### Como visualizar o projeto:
- Faça o download ou clone este repositório;
- Abra o arquivo index.html em qualquer navegador moderno.

### Tecnologias Utilizadas:
- HTML5 Semântico;
- CSS3 (Grid, Flexbox, Media Queries);
- JavaScript (DOM, Scroll Events);
- Swiper JS (Biblioteca de carrossel);
- Font Awesome (Ícones).
